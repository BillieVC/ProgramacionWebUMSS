La instrucción try le permite probar un bloque de código en busca de errores.

La declaración de captura le permite manejar el error.

La sentencia throw le permite crear errores personalizados.

La instrucción try le permite definir un bloque de código para que se pruebe en busca de errores mientras se ejecuta.

La instrucción catch le permite definir un bloque de código que se ejecutará, si ocurre un error en el bloque try.
